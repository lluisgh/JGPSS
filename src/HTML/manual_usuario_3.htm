Introduir blocs

 

Per poder introduir qualsevol bloc primer de tot s’ha de crear un procés, sino els botons es trobaràn deshabilitats.

Els blocs que es poden crear es troben a les pestanyes de la dreta.

Al introduïr les dades correctament i donar-li al botó ok en la pantalla del Model es veurà dibuixat el bloc introduït.

 

 

Introduir blocs si tenim més d’un procés

 

En el cas de que tinguem més d’un procés i es vulguin introduir blocs al model a diferents procesos, el que s’ha de fer per seleccionar el procés on es vol introduir el nou bloc en el combo que es troba a la dreta com es mostra en la següent figura.

 

Consultar Bloc

 

Per consultar les dades d’un bloc s’ha de clickar amb el botó dret sobre el bloc que es vol consultar les dades. I seleccionar l’opció “Bloc Properties”.

 

Al seleccionar aquesta opció surt, com es mostra en la figura següent, una pantalla amb les dades del bloc seleccionat.

 

 

Modificar Bloc

 

Per modificar les dades d’un bloc, com a consultar les dades d’un bloc, s’ha de clickar amb el botó dret sobre el bloc que es vol modificar i seleccionar l’opció “Modify Block”.

 

 

Al seleccionar aquesta opció surt la pantalla amb els valors del bloc per a modificar.

 

 

Si acceptem modificarem els valors que s’haguin canviat, si cancel·lem no passarà res.

Si es vol comprovar si les dades s’han modificat amb els valors desitjats, es pot consultar el bloc, com s’ha explicat anteriorment i veure si les dades són les correctes.

Esborrar Bloc

 

Per Esborrar un bloc, com per consultar i modificar un bloc, s’ha de clickar amb el botó dret sobre el bloc que es vol esborrar del model i seleccionar l’opció “Delete Block”.

 

 

Al seleccionar aquesta opció surt el missatge de si volem esborrar el bloc.

 

 

Si accceptem s’esborrarà el bloc del model, si cancel·lem no passarà res.
